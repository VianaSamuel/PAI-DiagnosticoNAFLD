Telas de Exibição de Imagens
Clique e arraste sobre a imagem para aplicar zoom. É possível também a navegação entre imagens via setas do teclado.

Tela Salvar ROIs
Para salvar uma ROI, primeiro clique no botão fígado, clique em um ponto na imagem, clique no botão rim, clique em outro ponto, clique em salvar.

IMPORTANTE!!!
Dependendo da pasta onde o projeto foi salvo, podem haver problemas de permissão. Tente trocar para uma pasta salva diretamente na raiz do disco (como por exemplo, C:/G11).
A pasta rois DEVE EXISTIR, senão encontramos erros. Nesse caso, crie uma pasta chamada "rois" no mesmo diretório do código.
